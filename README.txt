For each arch (x86/x64):

1. Download esd "install.esd"
2. Run "esd2wim.cmd"
3. Download "cu" and "ssu" for current %arch%
   - SSU https://portal.msrc.microsoft.com/en-us/security-guidance/advisory/ADV990001
   - CU https://support.microsoft.com/en-us/help/4464619
4. Run "run.cmd"
5. Grab and do whatever with "install.wim"
